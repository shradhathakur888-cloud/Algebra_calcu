Help & Instructions

How to Use

1. Enter your equation using the math keyboard or type directly

2. Select the variable you want to solve for (x, y, z, a, b)

3. Click "Solve" to get step-by-step solutions

4. Click "Graph" to visualize the equation

5. Use "Simplify" to simplify expressions

6. Use "Factor" to factor expressions

Trigonometric Functions

Basic: sin(x), cos(x), tan(x)
Inverse: asin(x), acos(x), atan(x)
Amplitude: 2*sin(x), 3*cos(x)
Frequency: sin(2x), cos(3x)
Phase Shift: sin(x+π/2)
Combined: sin(x)+cos(x)

Graphing Tips for Trig Functions

• Use X range -10 to 10 (approx -3π to 3π)

• Use Y range -5 to 5 for most trig functions

• tan(x) has vertical asymptotes at π/2, 3π/2, etc.

• Inverse functions have restricted domains

• Double-click graph to reset view
